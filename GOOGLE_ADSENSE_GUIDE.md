## 📋 如何获取 AdSense ID

### 步骤 1: 申请 AdSense 账户

1. 访问 Google AdSense
   👉 https://www.google.com/adsense/start/

2. 使用 Google 账户登录

3. 填写申请表：
   - 网站 URL: `https://your-app.vercel.app`
   - 内容语言: 选择您的语言
   - 接受条款和条件

4. 提交申请

### 步骤 2: 等待审核

**审核时间**: 通常 1-2 周

**审核要求**:
- 网站必须公开可访问
- 有足够的原创内容
- 遵守 AdSense 政策
- 良好的用户体验

### 步骤 3: 获取发布商 ID

审核通过后：

1. 登录 AdSense Dashboard
   👉 https://www.google.com/adsense/

2. 点击左侧 **账户** → **账户信息**

3. 找到 **发布商 ID**
   - 格式: `ca-pub-1234567890123456`
   - 这就是您的 AdSense ID

4. 复制这个 ID

### 步骤 4: 添加到环境变量

在 `.env.local` 中添加：
```env
NEXT_PUBLIC_GOOGLE_ADSENSE_ID=ca-pub-你的ID
```

在 Vercel 中添加相同的变量。

## 📊 AdSense 要求

### 流量要求

**建议**:
- 每天至少 100 访问量
- 有真实用户互动
- 有回访用户

## 💰 预期收入

### 收入取决于

- **流量**: 访问量越高，收入越高
- **点击率**: 通常 0.5% - 2%
- **每次点击费用**: 通常 $0.20 - $2.00
- **地理位置**: 发达国家收入更高

### 示例估算

```
100 访问/天:
  → 点击率 1%
  → 每次点击 $0.50
  → 月收入: ~$15

1,000 访问/天:
  → 点击率 1%
  → 每次点击 $0.50
  → 月收入: ~$150

10,000 访问/天:
  → 点击率 1%
  → 每次点击 $0.50
  → 月收入: ~$1,500
```

## 🚀 集成步骤

### 代码已准备就绪

您的应用已经包含 AdSense 集成代码（在 `app/layout.tsx`）:

```typescript
{process.env.NEXT_PUBLIC_GOOGLE_ADSENSE_ID && (
  <Script
    id="google-adsense"
    src={`https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=${process.env.NEXT_PUBLIC_GOOGLE_ADSENSE_ID}`}
    strategy="afterInteractive"
    crossOrigin="anonymous"
    async
  />
)}
```

### 只需添加 ID

1. 获得 AdSense 批准
2. 获取发布商 ID
3. 添加到 `.env.local`
4. 重新部署
5. 广告会自动显示

## 📝 AdSense 政策

### 必读文档

1. **AdSense 计划政策**
   https://support.google.com/adsense/answer/48182

2. **网站站长指南**
   https://support.google.com/webmasters/answer/35769

3. **无效流量政策**
   https://support.google.com/adsense/answer/16737

### 关键规则

**可以做**:
- ✅ 展示 Google 广告
- ✅ 鼓励用户浏览网站
- ✅ 优化广告位置

**不能做**:
- ❌ 点击自己的广告
- ❌ 要求用户点击广告
- ❌ 使用机器人增加点击
- ❌ 在同一页面放太多广告
